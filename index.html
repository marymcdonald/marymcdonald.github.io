Sundial is self-hosted, open-source cron job monitoring and management service that users can operate across one or multiple nodes.

It is a dockerized three-tiered application coupled with a binary executable that includes CLI scripts and a listening server that operates as a background process.

Built on Node.js using Express, React Native, PostgreSQL, pg-boss, and pkg.
